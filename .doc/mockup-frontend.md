**Table Features**:
- **STT Column**: Auto-incremented serial number for each row
- **Sortable Headers**: All columns clickable with ↕ indicator
  - Click once: Sort Ascending (↑)
  - Click again: Sort Descending (↓)
- **Column Resizing**: Drag column borders to adjust width
- **Vertical Dividers**: Subtle borders between columns for clarity
- **Responsive**: Columns auto-adjust on smaller screens

**Data Grid Columns**:
| Column | Type | Width | Sortable | Filter |
|:-------|:-----|:------|:---------|:-------|
| STT | Number | 50px | No | No |
| Mã ĐH | Text | 100px | Yes | Search |
| Khách hàng | Text | 150px | Yes | Search |
| Ngày tiệc | Date | 110px | Yes | Date Range |
| Tổng tiền | Currency | 110px | Yes | Range |
| Status | Badge | 100px | Yes | Dropdown |
```

## 🔄 INTERACTIONS

### Loading States
- **Skeleton loaders** cho danh sách (KHÔNG dùng spinner)
- Card placeholder với shimmer effect
- Table rows placeholder

### Animations
- **150ms** ease-out cho tất cả transitions
- Subtle fade-in khi load data
- Slide-up animation cho modals

### Hover Effects
- Row highlight: `#1F1F1F`
- Button scale: 1.02
- Card lift: 2px shadow
